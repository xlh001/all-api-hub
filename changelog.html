更新日志

本页记录面向普通用户的主要更新点（功能变化 / 体验优化 / 问题修复）。如需查看完整历史版本与更详细的技术变更，请访问 GitHub Releases。

3.6.0

  • 新功能:
    • 账号管理：支持一键启用/停用账号；停用后各种功能都将跳过该账号，便于账号失效后保留数据
    • 标签：新增全局标签管理，并同步优化相关界面与交互，方便对账号进行分类管理
    • 弹窗：在标题栏展示当前版本号，并提供直达本更新日志的入口
    • 快速导出：CCSwitch 导出支持选择上游模型，导出配置更贴近实际使用场景

3.5.2

  • 问题修复:
    • 金额显示：优化极小数值的展示策略，避免因精度/取整导致显示不符合预期

3.5.1

  • 新功能:
    • 账号管理：调整“自动签到”开关位置（移动到自定义签到 URL 之上），配置更直观
    • 界面：对话框标题图标移除渐变底色，视觉更清爽、统一
  • 问题修复:
    • 密钥列表：修复展开密钥详情时，关闭对话框会导致白屏的问题

3.5.0

  • 新功能:
    • 自动识别：增加“检测较慢”提示提醒和相关文档链接，帮助用户排查和解决问题
    • 外部签到批量打开：支持全部打开在新窗口中，便于批量关闭和减少干扰
  • 问题修复:
    • 外部签到批量打开：流程重构为在后台服务执行，确保在pop-up场景中可以正确打开所有站点

3.4.0

  • 新功能:
    • CLIProxy：增强模型映射配置，支持直接选择上游模型，便于更精确地进行模型映射
  • 问题修复:
    • API：确保访问密钥始终带 sk- 前缀，避免因格式不一致导致的识别/复制问题

3.3.0

  • 新功能:
    • 自动签到：账号识别增加“用户名”信息，方便多账号场景下区分账号
    • 外部签到：支持批量触发外部签到，减少逐个操作的步骤
    • 自动刷新：最小刷新间隔不再限制最大值，允许设置更大的最小间隔以控制刷新频率
  • 问题修复:
    • 剪贴板读取：收紧触发条件，减少在非复制场景下的误触发
    • 兑换助手：弹窗提示前先校验全部兑换码，减少无效兑换提示
    • 存储：写入操作加入写锁，提升并发写入时的数据一致性
    • 界面：调整“复制模型名称”的相关本地化文案

3.2.0

  • 新功能:
    • “模型列表”页面新增「接口可用性检测」（Beta），用于快速确认当前密钥在指定模型上是否可用（例如：文本生成、工具/函数调用、结构化输出（按 JSON 结构返回）、联网搜索（Grounding）等检测项目）
    • “模型列表”页面新增「CLI 工具兼容性检测」（Beta），模拟 Claude Code / Codex CLI / Gemini CLI 的工具调用流程，评估接口在这些工具里的兼容情况
    • “关于”页面新增「评分与下载」：自动识别当前商店来源（Chrome / Edge / Firefox），并提供一键评分与其它商店下载入口
  • 问题修复:
    • 站点刷新遇到 HTTP 错误时，健康状态会显示状态码与错误原因，便于定位问题
    • 在侧边栏模式下，不再显示“在侧边栏中打开”按钮，避免重复打开
