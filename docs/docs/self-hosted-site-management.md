# 自建站点管理 (New API / AxonHub / Claude Code Hub 等)

> 🧪 该功能旨在把最常用的渠道运营动作（建站、调参、同步）浓缩进插件内，让你不必频繁往返各个系统的后台。

## 支持的系统类型

All API Hub 深度适配了以下开源/自建 AI 分发系统：

| 系统类型 | 核心管理对象 | 特点 |
|----------|------------|------|
| **New API / DoneHub / Veloera** | 渠道 (Channel) | 经典的渠道管理，支持 55+ 种上游类型，具备分组、优先级与权重体系。 |
| **AxonHub** | 渠道 (Channel) | 高性能 AI 网关，支持 15+ 种渠道类型，界面简洁，配置高效。 |
| **Claude Code Hub** | 提供商 (Provider) | 专注于多厂商接入与弹性调度，提供商管理逻辑清晰，适配多种响应协议。 |
| **Octopus** | 渠道 (Channel) | 面向个人的轻量级聚合服务，支持 6 种主流渠道类型。 |

## 功能概述

- 📋 **统一列表视图**：一眼查看所有渠道/提供商的名称、类型、模型列表、优先级与状态。
- ✏️ **跨平台增删改查**：根据所选系统类型自动适配表单字段，无需手动转换参数。
- 🔄 **渠道迁移与同步**：支持将现有的账号/密钥直接导入为自建站点的渠道，并支持单渠道或批量模型同步。
- 🗑️ **安全批量操作**：支持批量启用、停用、删除，操作前会有确认弹窗。
- 🔌 **下游生态联动**：创建渠道后，可联动“密钥管理”一键导出到 CherryStudio、CC Switch 等。

## 配置指南

在使用管理功能前，需先在插件中完成对应后台的连接配置。

### 1. 进入配置页面
打开扩展设置页，在左侧菜单进入 **“基础设置”**，找到 **“自建站点管理”**。

### 2. 填写连接信息

| 配置项 | 说明 |
|------|------|
| **Base URL** | 你的自建系统后台地址（通常是网页访问地址）。 |
| **认证凭证** | **New API 系列**: 需要管理员令牌 (Admin Token) 与 用户 ID。<br>**AxonHub**: 管理员邮箱与密码。<br>**Claude Code Hub**: 管理员邮箱与密码。<br>**Octopus**: 用户名与密码。 |

### 3. 验证连接
点击 **“验证配置”**。验证成功后，对应系统的管理入口将自动解锁。

## 渠道管理操作指南

### 1. 列表操作
在设置页面顶部选择 **“自建站点管理”**（或直接点击基础设置里的 **“管理渠道”**）：

- **搜索与过滤**：支持按名称、类型、状态进行实时搜索。
- **快速切换系统**：如果你配置了多个自建系统，可以通过顶部的切换器快速跳转。
- **自定义列**：对于不支持“优先级”或“权重”的系统（如 Octopus），对应列会自动隐藏。

### 2. 创建或编辑渠道
1. 点击右上角 **“新增渠道”**。
2. 表单会自动根据当前系统类型调整：
   - **New API**: 提供丰富的渠道类型和分组配置。
   - **Claude Code Hub**: 需选择 Provider 类型（OpenAI Compatible, Claude, Gemini 等）。
   - **AxonHub**: 支持快速配置模型列表。
3. 点击保存后，系统会直接调用后台 API 完成同步。

### 3. 安全验证 (2FA / OTP)
当执行敏感操作（如查看渠道真实密钥）时，如果系统开启了二次验证，插件会弹出验证窗口。
- 详情请参阅：[New API 安全验证 (2FA / OTP)](./new-api-security-verification.md)

### 4. 渠道迁移 (Beta)
这是一个高级功能，允许你将 **“账号管理”** 里的现有账号快速转换为自建站点的渠道。


1. 在渠道列表页点击 **“渠道迁移”**。
2. 选择来源（已有的账号/书签）和目标（当前的自建站点）。
3. 插件会自动：
   - 提取 Base URL 和 API Key。
   - 尝试自动识别上游可用的模型列表。
   - 按照预设规则生成渠道名称。

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| 配置验证失败 | 请确认 Base URL 是否填写正确（是否带 `https://`），以及管理员权限是否有效。部分系统要求关闭两步验证或使用特定的 API 令牌。 |
| 列表加载缓慢 | 当渠道数量较多时（>100个），受限于后端 API 性能，加载可能需要几秒钟，请耐心等待。 |
| 无法同步模型 | 请确认自建站点的后端网络是否可以正常访问上游地址（如 OpenAI / Claude 官网）。 |
| 某些字段显示 Unknown | 这通常是因为自建站点的版本过新或过旧，返回了插件尚未适配的类型 ID。 |
