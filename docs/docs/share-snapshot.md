# 分享快照 (Share Snapshot)

> 分享快照功能允许你将“资产总览”或“单个账号状态”生成精美的卡片图片，以便一键分享到社区、社交媒体或发送给朋友。快照采用了隐私保护设计，会自动隐藏敏感信息（如 API Key）。

## 适合什么场景

- 你想向他人展示当前的资产总览，但又不想手动截图并抹除敏感数据。
- 在社区（如 [Linux.do](https://linux.do)）交流中，展示你的余额热力图或模型分布。
- 快速记录某个时刻的账号用量，作为备份图片保存。

## 如何生成快照

### 1. 分享总览快照 (Overview Snapshot)
1. 点击插件图标打开 **`总览 (Overview)`** 页面。
2. 在右上角（标题栏右侧）找到 **`分享总览快照`** 按钮（图标通常为一个相机或分享符号）。
3. 点击后，插件会根据当前的余额、今日用量、账号数量等生成一张网格背景的精美卡片。

### 2. 分享账号快照 (Account Snapshot)
1. 打开插件设置页，进入 **`账号管理`**。
2. 找到你想分享的账号卡片。
3. 点击操作菜单（三个点或右键），选择 **`分享账号快照`**。
4. 生成的快照将包含该账号的名称、站点地址、余额、今日收支以及当前的健康状态。

## 隐私保护机制

为了确保你的资产安全，分享快照遵循以下隐私原则：

- **自动脱敏**：绝不会在图片中显示完整的 API Key 或 Token 信息。
- **站点地址遮蔽**：仅显示站点的域名或名称，隐藏可能包含私有标识的完整路径。
- **本地生成**：图片完全在你的浏览器本地渲染生成，不会上传到任何服务器。

## 交互与保存

- **复制到剪贴板**：插件会优先尝试将生成的图片直接复制到剪贴板，你只需在聊天框执行 `Ctrl+V` (或 `Cmd+V`) 即可发送。
- **自动下载**：如果你的浏览器不支持直接复制图片到剪贴板，插件会自动下载一张 `.png` 图片到你的电脑。
- **标题文案复制**：快照下方通常会提供一键复制标题文案的按钮，方便你配合图片一起发布。

## 自定义风格 (实验性)

插件为快照内置了动态的 **Mesh Gradient (网格渐变)** 背景。每次生成的快照背景可能会有细微的色彩差异，确保每一张分享图都独一无二且具有现代感。
