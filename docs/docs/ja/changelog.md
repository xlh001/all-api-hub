# 更新履歴

## 2.39.0
- アカウントデータ更新時、アカウントサイトのチェックインサポート状況を自動で検出し、修正する機能を追加
- バージョン更新時、自動で更新履歴ページを開き、対応するバージョン番号のアンカーに移動する

## 2.38.0
- ピン留めされたアカウントのドラッグ＆ドロップによる並べ替え機能に対応
- 兌換アシスタントのキーボードナビゲーションに対応
  - キーボードの上下キーで具体的な兌換アカウントを直接選択
  - Enterキーを押して兌換を確定
- 一時的なシールド回避タブに、そのタブが本プラグイン由来であることと具体的な用途を説明するヒントを追加
- より良いシールド回避ウィンドウの表示方法（シングルウィンドウ・マルチタブ）。短時間のリクエストはすべて同じウィンドウを再利用し、干渉を最小限に抑える
- New-API サイトアカウントのチェックイン状態検出および自動チェックインに対応

## 2.37.0
- New-API チャネル管理のユーザーエクスペリエンスを最適化
- より柔軟な兌換コード形式の検出オプションに対応。カスタム兌換コード形式に遭遇した場合でも、正しく兌換コードを識別し、兌換アシスタントをポップアップ表示する
- いくつかの既知の問題を修正

## 2.36.0
