# 更新履歴

## 3.5.0
- **新機能:**
  - 自動認識：「検出が遅い」というヒントと関連ドキュメントリンクを追加し、ユーザーが問題の特定と解決を支援できるようにしました。
  - 外部チェックインの一括オープン：すべてを新しいウィンドウで開くことをサポートし、一括クローズや干渉の軽減を容易にしました。
- **不具合修正:**
  - 外部チェックインの一括オープン：処理をバックグラウンドサービスで実行するようにリファクタリングし、ポップアップのシナリオで全てのサイトが正しく開かれるようにしました。

## 3.4.0
- **新機能:**
  - CLIProxy：モデルマッピング設定を強化し、アップストリームモデルを直接選択できるようにして、より正確なモデルマッピングを容易にしました。
- **不具合修正:**
  - API：アクセスキーが常に `sk-` プレフィックスを持つようにし、形式の不一致による認識/コピーの問題を回避しました。

## 3.3.0
- **新機能:**
  - 自動チェックイン：アカウント識別に「ユーザー名」情報を追加し、複数アカウントのシナリオでアカウントを区別しやすくしました。
  - 外部チェックイン：外部チェックインの一括トリガーをサポートし、個別の操作ステップを削減しました。
  - 自動リフレッシュ：最小リフレッシュ間隔の最大値制限を解除し、リフレッシュ頻度を制御するために、より大きな最小間隔を設定できるようにしました。
- **不具合修正:**
  - クリップボード読み取り：トリガー条件を厳格化し、コピー以外のシナリオでの誤作動を減らしました。
  - 引き換えアシスタント：ポップアップヒントの前にすべての引き換えコードを検証し、無効な引き換えヒントを減らしました。
  - ストレージ：書き込み操作に書き込みロックを追加し、同時書き込み時のデータ一貫性を向上させました。
  - UI：「モデル名のコピー」に関連するローカライズされた文言を調整しました。

## 3.2.0
- **新機能:**
  - 「モデルリスト」ページに「インターフェース可用性検出」（Beta）を新設。現在のキーが指定されたモデルで利用可能か（例：テキスト生成、ツール/関数呼び出し、構造化出力（JSON構造での返却）、インターネット検索（Grounding）などの検出項目）を迅速に確認できます。
  - 「モデルリスト」ページに「CLIツール互換性検出」（Beta）を新設。Claude Code / Codex CLI / Gemini CLIのツール呼び出しプロセスをシミュレートし、これらのツールにおけるインターフェースの互換性を評価します。
  - 「概要」ページに「評価とダウンロード」を新設：現在のストアソース（Chrome / Edge / Firefox）を自動認識し、ワンクリック評価と他のストアのダウンロード入口を提供します。
- **不具合修正:**
  - サイトのリフレッシュ時にHTTPエラーが発生した場合、問題の特定を容易にするために、健康状態にステータスコードとエラー原因が表示されるようになりました。
  - サイドバーモードでは、「サイドバーで開く」ボタンが表示されなくなり、重複して開くことを回避します。
