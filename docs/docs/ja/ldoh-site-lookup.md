# LDOH サイト検索

> LDOH サイト検索機能は、 [Linux.do](https://linux.do) コミュニティのデータに基づいた拡張ツールです。追加したサイトのアドレスに基づいて、コミュニティ内の対応する議論、宣伝、または説明投稿を自動的にマッチングし、サイトの背景、信頼性、および使用フィードバックをより深く理解するのに役立ちます。

## なぜこの機能が必要なのか

AI 中継サイトのエコシステムでは、多くのサイトがコミュニティメンバーによって運営されています。 Linux.do の関連する議論に直接ジャンプすることで、以下が可能になります：
- **ユーザーレビューの確認**: サイトの安定性、レスポンス速度、アフターサポートについて知ることができます。
- **最新動向の把握**: サイトに新しいキャンペーン、モデルの調整、またはアドレスの変更があるかどうかを確認できます。
- **リスク回避の参考に**: サイトに対してコミュニティ内で否定的なフィードバックが多い場合、使用時の判断材料になります。

## 仕組み

1. **自動マッチング**: サイトを追加したり、既存のサイトの詳細を表示したりするときに、拡張機能はサイトの `Base URL` とバックエンドで管理されている [LDOH サイトインデックス](https://ldoh.105117.xyz) を自動的に照合します。
2. **インジケーターの表示**: マッチングに成功すると、 **`アカウント管理`** のサイト情報欄に関連リンクまたはジャンプアイコンが表示されます。
3. **ワンクリックでアクセス**: インジケーターをクリックすると、拡張機能は対応する Linux.do の検索リンクを生成し、関連するコミュニティの議論ページに直接誘導します。

## プライバシーとパフォーマンス

- **サイレントマッチング**: マッチングプロセスはバックグラウンドで自動的に実行され、余計なポップアップで邪魔をすることはありません。
- **キャッシュメカニズム**: ネットワークリソースを節約するため、マッチングインデックスはローカルに 12 時間キャッシュされます。拡張機能は、キャッシュが期限切れになった場合、または手動更新がトリガーされた場合にのみ、インデックスの更新をリクエストします。
- **ログイン不要**: この機能は、比較と検索のリダイレクトに公開 URL のみを使用します。 Linux.do アカウントにログインする必要はありません。

## 注意事項

- **マッチングしない場合**: コミュニティサイトは絶えず出現しているため、インデックスがすべてのサイトを 100% カバーしているとは限りません。サイトにマッチング結果が表示されない場合は、単にそのアドレスが現在の LDOH インデックスにまだ登録されていないことを意味します。
- **データソース**: この機能は、コミュニティのボランティアによって維持されている [LDOH サイトアグリゲーター API](https://ldoh.105117.xyz) によって提供されています。
