# キー管理

> キー管理（Key Management）機能を使用すると、追加したアカウント配下のすべての API キー（トークン）を一元的に表示・管理できます。New API 系サイトの場合、サイトの管理画面で作成したトークンを自動同期したり、ワンクリックでの修復、検証、エクスポートが可能です。

## 活用シーン

- 複数の中継サイトで大量のトークンを作成しており、各管理画面にログインすることなく、一箇所で検索や一括エクスポートを行いたい。
- 作成時に一度だけキーが表示され、その後は非表示になるサイトがある。拡張機能を使用して、これらの「見えない」キーを記録または「修復」したい。
- アカウント配下の特定のトークンがまだ有効か、あるいは CLI ツールと互換性があるかを確認したい。
- アカウントのトークンを [API 認証情報](./api-credential-profiles.md) に素早く保存し、より詳細なタグ付けや管理を行いたい。

## 機能の概要

- **トークンの一元リスト**: アカウントごとにグループ化、または全トークンを表示。名前やキーの内容で検索可能。
- **自動同期と更新**: サポートされているサイト（New API など）では、最新のトークンリストを自動的に取得。
- **キーの修復 (Repair)**: セキュリティ上の理由でフルキーが表示されないサイト（`sk-***...` と表示される）において、管理者権限を使用してフルキーを再取得または再生成。
- **ヘルスチェック**: トークンの残高、使用量、有効期限をリアルタイムで表示。
- **外部ツール連携**: 単一または複数のトークンを、主要な AI ツールへワンクリックでエクスポート。

## アクセス方法

1. 拡張機能の設定ページを開きます。
2. 左メニューから **`キー管理`** を選択します。

## 主な操作

### 1. トークンリストの自動更新
キー管理ページ上部の **`更新`** ボタンをクリックします。拡張機能は選択したアカウントのバックエンド API に接続し、最新のトークンとそのステータス（残高、使用量、期限）を取得します。

### 2. 欠落したキーの修復 (Repair)
一部のサイトでは、セキュリティのためにフルキーが非表示になっています。拡張機能にサイトの管理者権限や管理アカウントを設定している場合：
1. 上部の **`欠落したキーを修復`** をクリックします。
2. 拡張機能はリストをスキャンし、管理 API を通じて非表示のフルキーの補完を試みます。
3. 修復されたキーはブラウザのローカルストレージに安全に保存され、コピーやエクスポートに利用できるようになります。

### 3. 検証とステータス確認
各トークンカードには、現在の「ヘルスステータス」が表示されます：
- **残高/使用量**: サイトから取得したリアルタイムのクォータ情報。
- **API の検証**: 手動で API コールテストを実行し、キーが使用可能か確認します。
- **CLI 互換性の検証**: 特定の CLI 環境でトークンが正しく動作するかテストします。

### 4. 他のツールへのエクスポート
トークンの横にある **`エクスポート`** ボタンをクリックして、以下のツールに同期できます：
- CherryStudio
- CC Switch
- Kilo Code
- CLIProxyAPI
- Claude Code Router
- 設定済みの自炊・ホスト型管理サイト

## 認証情報との連携

特定のトークンが非常に便利であると感じたり、詳細なメモやタグを付けたい場合は、 **`API 認証情報に保存`** をクリックします。これにより、`URL + キー` のペアが [API 認証情報](./api-credential-profiles.md) モジュールにコピーされます。

## よくある質問

| 質問 | 回答 |
|------|------|
| なぜフルキーが表示されないのですか？ | サイトのバックエンドでキーの再表示が制限されている可能性があります。「修復」機能（適切な権限が必要）を試してください。 |
| どのサイトがサポートされていますか？ | 現在、New API とその派生（DoneHub、Veloera など）が最も完全にサポートされています。 |
| ここでトークンを削除すると、サイトからも削除されますか？ | いいえ。ここでの削除は拡張機能のローカルキャッシュからの削除のみであり、オンラインのアカウントには影響しません。 |
| キーの安全性はどうなっていますか？ | すべてのキーはブラウザのローカルストレージに暗号化されて保存されます。手動でエクスポートしたり WebDAV 同期を設定したりしない限り、デバイスの外に出ることはありません。 |
