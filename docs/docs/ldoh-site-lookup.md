# LDOH 站点查找 (Ldoh Site Lookup)

> LDOH 站点查找功能是一个基于 [Linux.do](https://linux.do) 社区数据的增强工具。它能根据你添加的站点地址，自动匹配社区中对应的讨论、推广或说明贴，帮助你更深入地了解该站点的背景、信誉与使用反馈。

## 为什么需要此功能

在 AI 中转站生态中，很多站点都是由社区成员运营的。通过直接跳转到 Linux.do 的相关讨论，你可以：
- **查看用户评价**：了解该站点的稳定性、响应速度和售后情况。
- **获取最新动态**：发现站点是否有新的活动、模型调整或地址变更。
- **避坑参考**：如果一个站点在社区中有较多负面反馈，可以作为你使用时的参考。

## 如何工作

1. **自动匹配**：当你添加一个站点或查看已添加站点的详细信息时，插件会自动将该站点的 `Base URL` 与后台维护的 [LDOH 站点索引](https://ldoh.105117.xyz) 进行比对。
2. **显示标识**：如果匹配成功，在 **`账号管理`** 的站点信息栏位，你会看到一个关联的链接或跳转图标。
3. **一键直达**：点击该标识，插件会为你自动生成对应的 Linux.do 搜索链接，带你直达该站点的相关讨论区。

## 隐私与性能

- **静默匹配**：匹配过程在后台自动执行，不会产生额外的弹窗干扰。
- **缓存机制**：为了节省网络资源，匹配索引会在本地缓存 12 小时。只有在缓存失效或手动刷新时，插件才会请求一次索引更新。
- **无需登录**：此功能仅通过公开的 URL 进行比对和搜索跳转，不要求你登录 Linux.do 账号。

## 相关说明

- **匹配不成功？**：由于社区站点层出不穷，索引可能无法覆盖 100% 的站点。如果你的站点没有显示匹配结果，这仅仅代表该地址尚未被收录到当前的 LDOH 索引中。
- **数据来源**：该功能由社区志愿者维护的 [LDOH 站点聚合 API](https://ldoh.105117.xyz) 提供支持。
