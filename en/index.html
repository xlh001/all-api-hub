All API Hub - AI Aggregation Proxy Manager

Open-source browser extension for unified management of third-party AI aggregation proxies and self-hosted New API instances: automatically identifies accounts, checks balances, syncs models, manages keys, and supports cross-platform and cloud backup.

Get StartedChrome Web StoreEdge Add-onsFireFox Add-ons

Intelligent Site Management

Automatically identifies AI aggregation proxy sites and creates access Tokens, intelligently retrieves site names and recharge Ratios, supports duplicate detection and manual addition.

Multi-Account System

Supports adding multiple accounts per site, account Grouping and quick switching, real-time balance checking, and detailed usage logs.

Token and Key Management

Conveniently manage all API Keys, supporting viewing, copying, refreshing, and batch operations.

Model Information Viewing

Clearly displays the list of supported models and pricing information for the site.

Check-in Status Monitoring

Automatically detects which sites support check-ins, marks accounts that haven't checked in today, allowing you to complete multi-site check-ins sequentially in one panel, reducing wasted free Quota due to forgotten check-ins.

Quick Export Integration

One-click export of configurations to CherryStudio and New API, simplifying the API usage process.

New API System Management

Supports Channel management and model list synchronization for self-hosted New API instances, and provides a dedicated Channel management interface.

Data Backup and Recovery

Supports JSON format import/export and WebDav cloud backup, enabling cross-device data synchronization.

Full Platform Support

Compatible with Chrome and Firefox browsers, supports mobile browsers like Kiwi Browser, and adapts to dark mode.

Privacy and Security

Runs completely offline, all data is stored locally, and all core features can be used without an internet connection.

Cloudflare Bypass Assistant

Automatically pops up to bypass the 5-second shield when encountered, ensuring the site can be identified and recorded.

Quick Export

One-click export of site configurations to CherryStudio, New API, and CC Switch.

Introduction

The AI ecosystem now features an increasing number of aggregation proxies and self-hosted panels based on the New API series. Managing the balance, model lists, and API keys for these various sites simultaneously is often fragmented and time-consuming.

All API Hub, as a browser extension, automatically identifies accounts on these sites, allowing one-click balance checking, model management, key management, and automatic check-ins. it also provides tools like model synchronization and Channel management for self-hosted New API instances. Currently, it supports proxy accounts based on the following projects:

  • one-api
  • new-api
  • Veloera
  • one-hub
  • done-hub
  • Sub2API (Basic functionality supported)
  • Neo-API (Closed source)
  • Super-API (Closed source)
  • RIX_API (Closed source, basic functionality supported)
  • VoAPI (Closed source, older versions supported)
