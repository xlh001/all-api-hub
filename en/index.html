All API Hub - AI Aggregation Relay Station Manager

Open-source browser extension to uniformly manage third-party AI aggregation relay stations and self-built New APIs: automatically recognize accounts, view balances, sync models, manage keys, and support cross-platform and cloud backup.

Get StartedChrome StoreEdge StoreFireFox Store

Intelligent Site Management

Automatically identify AI aggregation relay stations and create access tokens. Intelligently obtain site names and recharge ratios. Supports duplicate detection and manual addition.

Multi-Account System

Supports adding multiple accounts for each site, account grouping and quick switching, real-time balance viewing, and detailed usage logs.

Token and Key Management

Conveniently manage all API Keys, supporting viewing, copying, refreshing, and batch operations.

Model Information Viewing

Clearly displays the list of supported models and pricing information for the site.

Check-in Status Monitoring

Automatically detects which sites support check-ins and marks accounts that have not checked in for the day. This allows you to complete multi-site check-ins sequentially within a single panel, reducing the waste of free quota due to forgotten check-ins.

Quick Export for Integration

One-click export of configurations to Cherry Studio and New API, simplifying the API usage process.

New API Class System Management

Supports channel management and model list synchronization for self-built New API instances, and provides a dedicated channel management interface.

Data Backup and Recovery

Supports JSON format import/export and WebDAV cloud backup for cross-device data synchronization.

Full Platform Support

Compatible with Chrome, Firefox browsers, supports mobile browsers like Kiwi Browser, and adapts to dark mode.

Privacy and Security

Runs completely offline, all data is stored locally, and all core functions can be used without an internet connection.

Cloudflare Anti-Bot Assistant

Automatically pops up to bypass the five-second shield when encountered, ensuring sites can be identified and recorded.

Quick Export

One-click export of site configurations to Cherry Studio, New API, and CC Switch.

Introduction

The AI ecosystem now has an increasing number of aggregation relay stations and self-built panels based on the New API series. Managing the balances, model lists, and API keys of various sites simultaneously is often scattered and time-consuming.

All API Hub, as a browser extension, can automatically recognize accounts on these sites and provide one-click access to view balances, manage models, keys, and perform automatic check-ins. It also offers tools like model synchronization and channel management for self-built New APIs. Currently, it supports accounts from relay stations based on the following projects:

  • one-api
  • new-api
  • Veloera
  • one-hub
  • done-hub
  • Sub2API (Basic functionality support)
  • Neo-API (Closed source)
  • Super-API (Closed source)
  • RIX_API (Closed source, basic functionality support)
  • VoAPI (Closed source, older version support)
