All API Hub - Your All-in-One AI Asset Manager

Open-source browser extension to unify the management of third-party AI aggregation relays and self-built New APIs: automatically identify accounts, compare model prices, verify API/CLI compatibility, synchronize models and channels, and support cross-platform and encrypted WebDAV backups.

📦 Unified Asset Dashboard

Consolidate multiple sites and accounts into a single panel for a clear overview of balances, usage, and health status, with smart site identification.

🏷️ Independent API Credentials

Save Base URL + Key without a site account. Support tag filtering and reuse for model viewing and API validation.

💰 Model Price Comparison

Compare actual costs across different sites, automatically identify the best group, and find the most cost-effective path.

✅ In-depth API Validation

Batch test model availability, Token compatibility, and CLI proxy connectivity to troubleshoot integration issues.

📈 Detailed Usage Analytics

Analyze usage and costs by date, account, and model. Includes heatmaps, latency views, and slow request analysis.

📅 Auto Check-in Assistant

Centrally manage check-ins for supported sites. Includes automatic scheduled check-ins and custom URL jumps.

🚀 Fast Ecosystem Integration

Instant sync to CherryStudio, CC Switch, Kilo Code, or push accounts directly to your self-hosted management backend.

🛠️ Self-hosted Site Linkage

Deeply adapted for New API, AxonHub, Claude Code Hub, and more, enabling channel management and model sync.

🔒 Privacy & Secure Sync

Local-first storage with encrypted WebDAV sync. Includes a Cloudflare assistant to handle bot challenges automatically.

Introduction

In the AI era, many of us keep multiple relay-site accounts to save money or try different models. Managing them quickly becomes painful: balances are scattered, pricing is messy, and daily manual check-ins are easy to miss.

All API Hub is built to solve those problems. It is your AI asset center, making management simpler, clearer, and more automated.

🎯 Use Cases

👤 I am a regular AI user (recommended for beginners)

  • How do I start?: Download and install the extension -> Add my first account
  • I want to save money: Earn credits with auto check-in -> Compare model prices across sites
  • I want less manual work: See asset changes at a glance -> Sync accounts to other AI tools

🛠️ I am an advanced user (Key collector)

  • Key management: Save standalone URL+Key pairs as credentials
  • Availability testing: Batch verify API and CLI compatibility
  • Cross-device sync: Configure encrypted WebDAV backup

👑 I am a site admin (operator area)

  • Efficiency tools: Manage channels inside the extension -> Batch sync models
  • Configuration optimization: Set up model redirection
  • Security safeguards: Handle 2FA / OTP verification

🧩 Supported Site Architectures

All API Hub is deeply compatible with the following open-source and closed-source systems:

  • Open-source architectures: One API, New API, Sub2API, Veloera, AxonHub, One-Hub, Done-Hub, and more.
  • Specialized architectures: AIHubMix, AnyRouter, Neo-API, Super-API, VoAPI, and more.

If you use Safari on macOS, read the Safari Installation Guide first. If you use QQ Browser, 360 Browser, Brave, or similar browsers, read the Manual Installation Guide.
