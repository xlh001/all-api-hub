All API Hub - AI Aggregation Relay Manager

Open-source browser extension that automatically identifies and manages all AI aggregation relay accounts, view balances, synchronize models, manage keys, and supports cross-platform and cloud backup.

Get StartedChrome Web StoreEdge Add-onsFirefox Add-ons

Intelligent Site Management

Automatically identifies AI aggregation relay sites and creates access Tokens, intelligently retrieves site names and top-up Ratios, supports duplicate detection and manual addition.

Multi-Account System

Supports adding multiple accounts per site, account Grouping and quick switching, real-time balance viewing, and detailed usage logs.

Token and Key Management

Conveniently manage all API Keys, supporting viewing, copying, refreshing, and batch operations.

Model Information View

Clearly displays the list of models and Channel information supported by the site, supports automatic synchronization of models for New API and compatible systems.

Check-in Status Monitoring

Automatically detects if the site supports check-in functionality and displays the current check-in status.

Quick Export and Integration

One-click export of configurations to CherryStudio and New API, simplifying the API usage process.

Data Backup and Recovery

Supports JSON format import/export and WebDav cloud backup, enabling cross-device data synchronization.

Cross-Platform Support

Compatible with Chrome and Firefox browsers, supports mobile browsers like Kiwi Browser, and adapts to dark mode.

Privacy and Security

Runs completely offline, all data stored locally, no internet connection required to use all core features.

Introduction

Currently, there are numerous AI aggregation relay sites on the market, and checking information such as balances, model lists, and keys requires logging into each one individually, which is very tedious.

This extension can automatically identify and integrate management for AI aggregation relay accounts based on the following projects:

  • one-api
  • new-api
  • Veloera
  • one-hub
  • done-hub
  • Neo-API (closed source)
  • Super-API (closed source)
  • RIX_API (closed source, basic functionality supported)
  • VoAPI (closed source, older versions supported)
