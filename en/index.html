All API Hub - AI Aggregation Proxy Manager

Open-source browser extension that automatically identifies and manages all AI aggregation proxy accounts, views balances, synchronizes models, manages keys, and supports cross-platform and cloud backup.

Get StartedChrome Web StoreEdge Add-onsFireFox Add-ons

Smart Site Management

Automatically identifies AI aggregation proxy sites and creates access Tokens, intelligently retrieves site names and top-up Ratios, supports duplicate detection and manual addition.

Multi-Account System

Supports adding multiple accounts per site, account Grouping and quick switching, real-time balance viewing, and detailed usage logs.

Token and Key Management

Conveniently manage all API Keys, supporting viewing, copying, refreshing, and batch operations.

Model Information View

Clearly displays the list of supported models and pricing information for each site.

Check-in Status Monitoring

Automatically detects whether a site supports check-in functionality and displays the current check-in status.

Quick Export Integration

One-click export of configurations to CherryStudio and New API, simplifying the API usage process.

New API-like System Management

Supports manual execution, model whitelist filtering, and automatic redirection generation, providing a dedicated Channel management interface.

Data Backup and Recovery

Supports JSON format import/export and WebDav cloud backup for cross-device data synchronization.

Full Platform Support

Compatible with Chrome, Firefox browsers, supports mobile browsers like Kiwi Browser, and adapts to dark mode.

Privacy and Security

Runs completely offline, all data is stored locally, and all core features can be used without an internet connection.

Cloudflare Bypass Assistant

Automatically bypasses Cloudflare's 5-second challenge with a pop-up, ensuring sites can be identified and recorded.

Quick Export

One-click export of site configurations to CherryStudio, New API, and CC Switch.

Introduction

There are numerous AI aggregation proxy sites on the market, and checking balances, model lists, and keys for each one individually is very cumbersome.

This extension can automatically identify and integrate the management of AI aggregation proxy accounts based on the following projects:

  • one-api
  • new-api
  • Veloera
  • one-hub
  • done-hub
  • Neo-API (closed source)
  • Super-API (closed source)
  • RIX_API (closed source, basic functionality supported)
  • VoAPI (closed source, older versions supported)
