All API Hub - AI Aggregation Gateway Manager

Open-source browser extension for unified management of third-party AI aggregation gateways and self-built New APIs: automatically recognizes accounts, checks balances, syncs models, manages keys, and supports cross-platform and cloud backups.

Get StartedChrome StoreEdge StoreFireFox Store

Intelligent Site Management

Automatically recognizes AI aggregation gateway sites and creates access tokens. Intelligently obtains site names and recharge ratios. Supports duplicate detection and manual addition.

Multi-Account System

Supports adding multiple accounts per site, account grouping, and quick switching. Real-time balance and detailed usage logs.

Token and Key Management

Conveniently manage all API Keys, supporting viewing, copying, refreshing, and batch operations.

Model Information Display

Clearly displays the list of supported models and pricing information for each site.

Check-in Status Monitoring

Automatically detects which sites support check-ins and marks accounts that haven't checked in for the day. This allows you to complete multi-site check-ins sequentially within a single panel, reducing the waste of free Quota due to forgotten check-ins.

Quick Export for Integration

One-click export of configurations to CherryStudio, CC Switch, Kilo Code, CLIProxyAPI, Claude Code Router, and self-built hosted sites.

Self-Built Site Backend Integration

Supports backend integration and Channel-related operations for self-built New API, DoneHub, Veloera, and Octopus instances.

Data Backup and Restore

Supports JSON format import/export and WebDAV cloud backup for cross-device data synchronization.

Cross-Platform Support

Compatible with browsers like Chrome, Edge, Firefox, and also usable on mobile browsers such as Edge for Mobile, Firefox for Android, Kiwi, etc. Supports dark mode.

Privacy and Security

Runs completely offline. All data is stored locally, and all core functions can be used without an internet connection.

Cloudflare Challenge Solver

Automatically prompts to solve 5-second challenges when encountered, ensuring sites can be recognized and recorded.

Introduction

The AI ecosystem now has an increasing number of aggregation gateways and self-built panels based on the New API series. Managing the balances, model lists, and API keys of various sites simultaneously is often scattered and time-consuming.

All API Hub, as a browser extension, can automatically recognize accounts on these sites and provide one-click access to view balances, manage models, keys, and perform automatic check-ins. It also offers backend integration and Channel-related tools for self-built New API, DoneHub, Veloera, and Octopus. Currently, it supports aggregation gateway accounts based on the following projects:

  • one-api
  • new-api
  • Veloera
  • one-hub
  • done-hub
  • Sub2API
  • AnyRouter
  • WONG Public Welfare Site
  • Neo-API (Closed Source)
  • Super-API (Closed Source)
  • RIX_API (Closed Source, basic functionality supported)
  • VoAPI (Closed Source, older versions supported)
