All API Hub - AI 統合ゲートウェイマネージャー

サードパーティのAI統合ゲートウェイや自社構築のNew APIを統一管理するオープンソースブラウザ拡張機能：アカウントの自動認識、残高表示、モデル同期、キー管理をサポートし、クロスプラットフォームとクラウドバックアップにも対応

使用開始Chrome ウェブストアEdge ウェブストアFireFox ウェブストア

スマートサイト管理

AI統合ゲートウェイサイトを自動認識し、アクセス トークンを作成します。サイト名とチャージ倍率をインテリジェントに取得し、重複検出と手動追加をサポートします。

マルチアカウントシステム

各サイトに複数アカウントを追加でき、アカウントのグループ化と高速切り替え、残高と詳細な使用ログのリアルタイム表示をサポートします。

トークンとキーの管理

すべてのAPIキーを簡単に管理でき、表示、コピー、リフレッシュ、一括操作をサポートします。

モデル情報表示

サイトがサポートするモデルリストと価格情報を明確に表示します。

チェックイン状態監視

チェックインをサポートするサイトを自動検出し、その日にまだチェックインしていないアカウントをマークします。これにより、1つのパネルで複数のサイトのチェックインを順番に完了でき、チェックイン忘れによる無料額面の無駄遣いを減らすことができます。

高速エクスポート統合

設定をCherryStudioとNew APIにワンクリックでエクスポートし、APIの使用プロセスを簡素化します。

New API クラスシステム管理

自社構築のNew APIインスタンスのチャネル管理とモデルリスト同期をサポートし、専用のチャネル管理インターフェースを提供します。

データバックアップと復元

JSON形式でのインポート/エクスポートとWebDAVクラウドバックアップをサポートし、クロスデバイスでのデータ同期を実現します。

全プラットフォーム対応

Chrome、Edge、Firefoxなどのブラウザと互換性があり、モバイル/スマートフォンブラウザ（例：Mobile Edge、Firefox for Android、Kiwiなど）でも使用でき、ダークモードにも対応しています。

プライバシーとセキュリティ

完全オフラインで動作し、すべてのデータはローカルに保存されます。インターネット接続なしで、すべてのコア機能を使用できます。

Cloudflare保護回避アシスタント

5秒ルールに遭遇した場合、自動的にポップアップして保護を回避し、サイトが認識および記録されることを保証します。

高速エクスポート

サイト設定をCherryStudio、New API、CC Switchにワンクリックでエクスポートします。

紹介

現在、AIエコシステムには、New APIシリーズに基づいた統合ゲートウェイや自社構築パネルがますます増えています。各サイトの残高、モデルリスト、APIキーを同時に管理することは、しばしば分散的で時間がかかります。

All API Hubはブラウザ拡張機能として、これらのサイトのアカウントを自動的に認識し、残高の表示、モデルやキーの管理、自動チェックインをワンクリックで行うことができます。また、自社構築のNew API向けに、モデル同期やチャネル管理などのツールを提供します。現在、以下のプロジェクトに基づいた統合ゲートウェイアカウントをサポートしています。

  • one-api
  • new-api
  • Veloera
  • one-hub
  • done-hub
  • Sub2API
  • AnyRouter
  • Neo-API（クローズドソース）
  • Super-API（クローズドソース）
  • RIX_API（クローズドソース、基本機能サポート）
  • VoAPI（クローズドソース、旧バージョンサポート）
