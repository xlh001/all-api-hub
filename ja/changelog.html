更新履歴

3.5.1

  • 新機能:
    • アカウント管理：「自動チェックイン」スイッチの位置を調整（カスタムチェックインURLの上に移動）、より直感的な設定に
    • インターフェース：ダイアログのタイトルアイコンからグラデーションの背景色を削除、よりすっきりとした統一感のあるビジュアルに
  • 不具合修正:
    • 秘密鍵リスト：秘密鍵の詳細を展開した際に、ダイアログを閉じるとホワイトスクリーンになる問題を修正

3.5.0

  • 新機能:
    • 自動認識：「検出が遅い」というヒントと関連ドキュメントへのリンクを追加し、ユーザーが問題を特定して解決するのを支援
    • 外部チェックインの一括オープン：すべてのサイトを新しいウィンドウで開くことをサポートし、一括クローズや干渉の軽減を容易に
  • 不具合修正:
    • 外部チェックインの一括オープン：ポップアップシナリオですべてのサイトが正しく開かれるように、バックグラウンドサービスで実行されるようにフローを再構築

3.4.0

  • 新機能:
    • CLIProxy：モデルマッピング設定を強化し、アップストリームモデルを直接選択できるようになり、より正確なモデルマッピングを容易に
  • 不具合修正:
    • API：アクセスキーが常に sk- プレフィックスを持つことを保証し、フォーマットの不一致による認識/コピーの問題を回避

3.3.0

  • 新機能:
    • 自動チェックイン：アカウント認識に「ユーザー名」情報を追加し、複数アカウントのシナリオでアカウントを区別しやすく
    • 外部チェックイン：外部チェックインの一括トリガーをサポートし、個別の操作ステップを削減
    • 自動更新：最小更新間隔の最大値制限をなくし、更新頻度を制御するために大きな最小間隔を設定できるように
  • 不具合修正:
    • クリップボード読み取り：トリガー条件を厳しくし、非コピーシナリオでの誤トリガーを削減
    • 兌換アシスタント：ポップアップ表示前にすべての兌換コードを検証し、無効な兌換のヒントを削減
    • ストレージ：書き込み操作に書き込みロックを追加し、同時書き込み時のデータ一貫性を向上
    • インターフェース：「モデル名をコピー」に関連するローカライズされたテキストを調整

3.2.0

  • 新機能:
    • 「モデルリスト」ページに「API可用性検出」（ベータ版）を新設し、指定されたモデルで現在の秘密鍵が利用可能か（例：テキスト生成、ツール/関数呼び出し、構造化出力（JSON構造で返す）、ネットワーク検索（Grounding）などの検出項目）を迅速に確認できるように
    • 「モデルリスト」ページに「CLIツール互換性検出」（ベータ版）を新設し、Claude Code / Codex CLI / Gemini CLI のツール呼び出しフローをシミュレートし、これらのツールでのAPIの互換性を評価
    • 「概要」ページに「評価とダウンロード」を新設：現在のストアソース（Chrome / Edge / Firefox）を自動認識し、ワンクリック評価と他のストアからのダウンロード入口を提供
  • 不具合修正:
    • サイト更新時にHTTPエラーが発生した場合、健全性ステータスにステータスコードとエラー原因が表示され、問題の特定を容易に
    • サイドバーモードでは、「サイドバーで開く」ボタンが表示されなくなり、重複して開くことを回避

3.1.1

  • 不具合修正:
    • サイトアクセス権限を拡大（host_permissions: <all_urls>）し、権限不足によるCookie取得失敗を軽減
