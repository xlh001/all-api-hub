シェアスナップショット

シェアスナップショット機能を使用すると、「資産の概要」や「個別アカウントのステータス」を美しいカード形式の画像として生成し、コミュニティや SNS、友人にワンクリックで共有できます。スナップショットはプライバシーに配慮して設計されており、API キーなどの機密情報は自動的に非表示になります。

活用シーン

  • 現在の資産概要を他人に示したいが、手動でスクリーンショットを撮って機密データを消去する手間を省きたい。
  • コミュニティ（Linux.do など）での交流において、残高ヒートマップやモデル分布を披露したい。
  • 特定の瞬間のアカウント使用状況を、参照用画像として素早く記録したい。

スナップショットの生成方法

1. 概要スナップショットを共有

  1. 拡張機能のアイコンをクリックして 概要 ページを開きます。
  2. 右上（タイトルバーの右側）にある 概要スナップショットを共有 ボタン（カメラまたは共有のアイコン）を探します。
  3. クリックすると、現在の残高、今日の使用量、アカウント数などに基づいた、メッシュグラデーション背景の美しいカード画像が生成されます。

2. アカウントスナップショットを共有

  1. 拡張機能の設定ページを開き、 アカウント管理 に移動します。
  2. 共有したいアカウントのカードを探します。
  3. 操作メニュー（三点リーダーまたは右クリック）をクリックし、 アカウントスナップショットを共有 を選択します。
  4. 生成されるスナップショットには、アカウント名、サイトアドレス、残高、今日の収支、および現在のヘルスステータスが含まれます。

プライバシー保護メカニズム

資産の安全を確保するため、シェアスナップショットは以下のプライバシー原則に従います：

  • 自動マスキング: フル API キーやトークン情報が画像に表示されることはありません。
  • サイトアドレスの隠蔽: サイトのドメインまたは名称のみが表示され、プライベートな識別子を含む可能性のあるフルパスは隠されます。
  • ローカル生成: 画像はブラウザ内で完全にローカルにレンダリング・生成されます。いかなるサーバーにもアップロードされることはありません。

操作と保存

  • クリップボードにコピー: 拡張機能は、生成された画像を直接クリップボードにコピーすることを優先します。チャットボックスなどで Ctrl+V (または Cmd+V) を実行するだけで送信できます。
  • 自動ダウンロード: ブラウザがクリップボードへの直接コピーをサポートしていない場合、自動的に .png 画像がダウンロードされます。
  • テキストタイトルのコピー: スナップショットの下部には通常、テキストタイトルをワンクリックでコピーできるボタンが用意されており、画像と一緒に投稿する際に便利です。

カスタムスタイル (実験的)

スナップショットには、動的なメッシュグラデーション背景が組み込まれています。生成されるたびに背景の色彩が微妙に異なり、共有するすべての画像がモダンでユニークなものになります。
