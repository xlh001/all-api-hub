モデルリストと価格比較

すべての中継サイトのモデル詳細を一目で把握。サイトをまたいだ価格比較、最適なグループの自動選択、一括可用性検証をサポートし、最もお得な選択を支援します。

機能の概要

  • 💰 リアルタイム価格比較: 課金倍率、グループ倍率、為替レートに基づいて、異なるサイトの各モデルの実際の単価を自動計算します。
  • 🎯 最適なパスの推奨: 「すべてのアカウント」ビューでは、システムが自動的にそのモデルの「全サイト最安値」または現在のサイトの「最適グループ」をマークします。
  • 🔍 多次元の詳細フィルタリング: サイトソース、API認証情報、プロバイダー（OpenAI、Anthropicなど）、課金モード、タグによる組み合わせフィルタリングをサポート。
  • ✅ 一括可用性検証: モデルの接続テスト、トークンの互換性チェック、CLIプロキシの可用性検証をサポート。
  • 📊 課金の透明化: 「トークン課金（Tokens）」と「リクエスト課金（Per Call）」を明確に区別し、公式の倍率比較を提供します。

アクセス方法

  1. 拡張機能のポップアップの左下にある 「モデルリスト」 アイコンをクリックします。
  2. または、 設定 → モデル管理 からアクセスします。

コア操作ガイド

1. データソースの選択

ページ上部のセレクターで、異なるデータソースを切り替えることができます：

  • すべてのアカウント: 追加したすべてのサイトアカウントを集計し、サイト間比較を有効にします。
  • 特定のアカウント: 指定したサイトの完全なモデルカタログを表示します。
  • API認証情報: 独立したAPI認証情報（サイトアカウント不要）でサポートされているモデルを表示します。

2. インテリジェントな価格比較

モデルカードには、モデルの詳細な課金情報が表示されます：

  • 課金倍率: サイトのモデル倍率（例：1.5）と補完倍率（例：2）を表示します。
  • グループ倍率: 現在のグループ（例：default、VIP）の追加倍率を表示します。
  • 実際の単価: システムが自動的に 1Mトークンあたり (USD/CNY) または 1回のリクエストあたり の価格に換算します。
  • 最安値バッジ: 「すべてのアカウント」ビューで、価格が最も安いサイトには緑色のバッジが表示されます。

3. グループの切り替えとシミュレーション

サイトが複数のユーザーグループ（例：default、vip、svip）をサポートしている場合、フィルターバーでグループを切り替えて、異なるティアのリアルタイムの価格変化をプレビューできます。

💡 ヒント: システムはデフォルトで、そのモデルに対して最も費用対効果の高い利用可能なグループを選択します。

4. モデルの検証

モデルカードの右側にある検証アイコンをクリックします：

  • モデルの検証: 軽量なリクエストを送信して、モデルが現在本当に利用可能かどうかを確認します。
  • CLI互換性の検証: モデルがストリーミング出力をサポートしているか、コマンドラインツールから正常に呼び出せるかをテストします。
  • 一括検証: ツールバーの 「一括検証」 をクリックすると、現在フィルタリングされているすべてのモデルのテストをキューに入れます。また、チェックボックスを使用して複数のモデルを手動で選択し、選択したセットに対してのみ検証を実行することもできます。

よくある質問

質問解決策
なぜ一部のモデルの価格が 0 と表示されるのですか？そのサイトがこのモデルを無料に設定しているか、サイトが有効な課金データを提供していない可能性があります。その場合、デフォルトのカタログにフォールバックします。
サイト間比較の為替レートはどのように計算されますか？拡張機能には、比較基準を統一するために 1 USD = 7.3 CNY の固定レート（New APIの慣例を参照）が組み込まれています。
「最適グループ」とはどういう意味ですか？モデルが複数のグループ（例：default と vip）で利用可能な場合、拡張機能は自動的に倍率が最も低いグループを選択して価格を計算します。
検証結果が「不明」と表示されるのはなぜですか？APIキーにそのモデルを呼び出す権限があるか、またはサイトが現在レート制限（Rate Limit）にかかっていないか確認してください。
