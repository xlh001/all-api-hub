All API Hub - 你的全能 AI 资产管家

开源浏览器插件，统一管理第三方 AI 聚合中转站与自建 New API：自动识别账号、比对模型价格、验证 API/CLI 兼容性、同步模型与渠道，并支持跨平台与加密 WebDAV 备份

🚀 开始使用Chrome 商店Edge 商店FireFox 商店Safari 安装

📦 资产统一看板

把多个站点与账号集中在一个面板里，余额、用量与健康状态一眼看清，支持智能识别地址自动添加。

🏷️ API 凭据库

无需账号，直接保存 Base URL + API Key，用于快速复制、验证接口、查看模型与余额/用量。

💰 模型价格比对

跨站对比各模型实际折合单价，自动锁定当前最优分组，助你寻找最实惠的调用路径。

✅ 深度接口验证

批量测试模型可用性、Token 兼容性及 CLI 代理连通性，轻松排查“站点能通、工具报错”的问题。

📈 用量深度统计

按日期、账号、模型多维分析用量与花费，提供热力图、延迟视图与慢请求分析。

📅 自动签到辅助

集中识别并处理支持签到的站点，支持定时自动签到、自定义 URL 与充值/兑换页面跳转。

🚀 极速生态集成

一键同步到 CherryStudio、CC Switch、Kilo Code 等，或直接将账号推送到自建管理后台。

🛠️ 自建站点联动

深度适配 New API、AxonHub、Claude Code Hub 等，实现渠道管理、模型同步与重定向。

🔒 隐私与安全同步

默认本地存储，支持加密 WebDAV 自动同步与备份，遇到 Cloudflare 挑战时可自动协助过盾。

介绍

在 AI 时代，为了省钱或体验不同模型，我们往往拥有多个中转站账号。但管理起来却很头疼：余额分散、价格混乱、每天手动签到太累...

All API Hub 为解决这些问题而生。 它是你的 AI 资产中心，让管理变得简单、直观且自动化。

🎯 你的使用场景

👤 我是普通 AI 用户 (新手推荐)

  • 我该怎么用？：下载并安装扩展 -> 添加第一个账号
  • 我想省钱：自动签到获取额度 -> 跨站模型价格比对
  • 我想更省事：资产变动一眼看清 -> 同步账号到其它 AI 工具

🛠️ 我是进阶玩家 (Key 收藏家)

  • 密钥管理：将独立 URL+Key 保存到 API 凭据库
  • 可用性测试：批量验证接口与 CLI 兼容性
  • 跨端同步：配置 WebDAV 加密备份

👑 我是站点管理员 (站长专区)

  • 效率工具：在插件内直接管理渠道 -> 批量同步模型
  • 配置优化：设置模型重定向
  • 安全保障：处理 2FA / OTP 验证

🧩 支持的系统架构

不论你用的是哪种架构，我们基本都支持：

  • 账号站点兼容架构：One API, New API, Veloera, One-Hub, Done-Hub, Sub2API 等。
  • 特色账号平台与兼容实现：AIHubMix, AnyRouter, Neo-API, Super-API, v-api 等。
  • 自建管理后台：New API, Veloera, Done-Hub, Octopus, AxonHub, Claude Code Hub 等，用于渠道管理、迁移和部分模型同步。
