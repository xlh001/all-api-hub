All API Hub - 中转站管理器

一个开源的 AI 中转站账号管理插件

开始使用Chrome 商店Edge 商店FireFox 商店

站点自动识别

自动识别并连接各类 AI 中转站点，一键创建并添加访问令牌（Token），简化初始设置流程。

充值比例识别

自动识别并显示中转站的充值比例，帮助您清晰地评估和最大化资金利用率。

多账号支持

支持在每个中转站点下添加和管理多个账号，支持账号分组与切换，轻松满足您的多账户使用与隔离需求。

余额与日志查询

实时查询和展示各账号的余额信息，并提供详细的使用日志，让您全面掌握消费动态。

令牌(Key)管理

提供便捷的令牌（Key）管理功能，包括查看、复制和刷新，确保操作的安全与高效。

模型与渠道信息

清晰展示站点所支持的全部模型及其关联渠道的详细信息，助您做出最佳的模型选择。

数据导入导出

支持 JSON 格式的数据备份与恢复，确保数据安全。

插件无需联网

所有核心功能均可在本地离线运行，无需联网，有效保护您的数据隐私并保障使用稳定性。

快速导出站点

支持一键将站点API配置导出到 CherryStudio 和 New API，简化API使用流程。

站点签到状态检测

支持检测站点是否支持签到和签到状态。

站点重复检测

在添加站点时自动进行重复检测，避免列表中出现重复的站点，保持数据整洁。

智能获取站点名称

能够智能识别并获取站点的真实名称，而非简单显示域名，提升了可读性。

WebDav 数据备份与恢复

支持通过 WebDav 备份和恢复所有账号与设置，保障数据安全并实现跨设备同步。

移动端支持

支持在移动设备上通过 Kiwi 浏览器或手机版 Firefox 使用，完整功能体验。

手动添加站点

支持手动添加任意中转站点，解决了自动识别失败时无法添加的问题，增强了灵活性。

深色模式

支持根据系统主题自动切换深色模式。

Firefox 支持

插件现已完全兼容 Firefox 浏览器，为火狐用户提供无缝的使用体验。

插件无需联网

所有核心功能均可在本地离线运行，无需联网，有效保护您的数据隐私并保障使用稳定性。

介绍

目前市面上有太多 AI-API 中转站点，每次查看余额和支持模型列表等信息都非常麻烦，需要逐个登录查看。

本插件可以便捷的对基于以下项目的AI 中转站账号进行整合管理：

  • one-api
  • new-api
  • Veloera
  • one-hub
  • done-hub
  • VoAPI
  • Super-API
